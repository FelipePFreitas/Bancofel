# Bancofel Web

Interface Angular responsiva para os endpoints do Bancofel. A API deve estar disponível em `http://localhost:8080`.

```bash
cd frontend
npm install
npm start
```

Abra `http://localhost:4200`. O back-end já permite CORS para essa aplicação.

## Operações conectadas

- Clientes PF/PJ: cadastrar, consultar, editar, desativar e reativar;
- Conta: consultar saldo e cadastrar chave Pix;
- Transações: depósito, saque, transferência e Pix.
